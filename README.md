# Convert-CSV-into-VCF
A csv file including contacts information (Names &amp; Mobile Numbers) convert into a vCard.  That vCard can be used to import multiple contact numbers it into Android Contacts, iPhone Contacts (via iCloud import), Outlook, etc. This is done by Python libraries.
